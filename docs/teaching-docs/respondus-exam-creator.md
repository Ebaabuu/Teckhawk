# Respondus Exam Creator - Access and Installation

## Summary

Guidance for instructors on installing the Windows-based tool for creating and publishing Canvas exams offline.

## Access

KU faculty and staff can download the software from the KU Software Store.

## Installation

1. Add "Respondus Exam Creator" to your cart and download the .exe.
2. During setup, you will be prompted for an Institution Name and Installation Password, which are found in the "My Software" section of the store.

## OS Note

This software is Windows only. Mac users must use a Windows emulator.
