# Respondus LockDown Browser - Installation

## Summary

Installation steps for students required to use the secure browser for online exams.

## Download

Students must use the unique KU-specific download link provided by their instructor or found in the KU IT Knowledge Base.

## Usage

- Once installed, locate the LockDown Browser icon on your desktop (Windows) or in Applications (macOS).
- Some Canvas exams require you to launch the browser after logging into Canvas in a standard browser.

## iPad

If permitted by the instructor, the app can be downloaded from the Apple App Store.
