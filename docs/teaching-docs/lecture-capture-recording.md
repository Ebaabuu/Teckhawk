# Lecture Capture - Requesting a Recording

## Summary

Protocols for instructors to request automated or ad-hoc recording of classroom lectures using Kaltura.

## Account Setup

To schedule a recording, instructors must first complete the KU Lecture Capture Request Form.

## Campus Support

- **Lawrence:** Media Production Studio (itmps@ku.edu | 785-864-6452)
- **Edwards:** Educational Technology (kuec_edtech@ku.edu | 913-897-8719)

## Self-Service

Instructors can also use Kaltura Capture for personal/desktop recording on their own workstations.
