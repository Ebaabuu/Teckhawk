# BitLocker - Finding Recovery Keys on Personal Devices

## Summary

This article provides guidance for users who need to locate a BitLocker recovery key for their personal Windows devices.

## Key Information

For personal devices, BitLocker recovery keys are typically stored in the user's personal Microsoft account.

## Steps

1. Log in to your [Microsoft Account Recovery Key page](https://account.microsoft.com/devices/recoverykey).
2. Identify the device and the corresponding Recovery Key ID.
3. Enter the 48-digit recovery key on your device when prompted.
