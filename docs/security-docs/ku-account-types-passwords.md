# KU Account - Types of Accounts and Managing Passwords

## Summary

Overview of the different account types at KU (Standard Online ID, Admin, STA, and Jayhawk Global).

## Password Changes

Performed at [myidentity.ku.edu/services/password/change](https://myidentity.ku.edu/services/password/change).

## Manual Reset

If self-service fails, an administrative reset requires a government-issued ID shown via video call or in-person at Anschutz Library.
