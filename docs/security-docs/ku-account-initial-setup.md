# KU Account - Setting Up KU OnlineID Account

## Summary

The initial setup process for new students and employees.

## Link

[myidentity.ku.edu/setup](https://myidentity.ku.edu/setup)

## Requirements

You will need your 7-digit KU ID number and date of birth.

## Security

During setup, users will create a password and are prompted to immediately configure Duo MFA.
