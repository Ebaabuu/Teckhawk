# Jayhawk Global - Account Setup

## Summary

Setup process for new Jayhawk Global users to access KU resources like Canvas.

## Process

1. Enroll in a course via the Jayhawk Global portal.
2. Wait for an email from `itcsc@ku.edu` containing your Online ID and a setup link.
3. Visit the setup link, enter your ID and date of birth, and create a password.

## Support

Contact `jayhawkglobal@ku.edu` for enrollment issues.
