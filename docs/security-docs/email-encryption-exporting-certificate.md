# Email Encryption - Exporting InCommon Email Certificate

## Summary

Steps to export your InCommon email encryption certificate for backup or installation on other devices.

## Windows

1. Open **Control Panel** > **Internet Options** > **Content** > **Certificates**.
2. Select the InCommon certificate and click **Export**.
3. Ensure "Yes, export the private key" is selected.

## macOS

1. Open **Keychain Access**.
2. Locate the InCommon certificate under **Certificates**.
3. Right-click and select **Export**.

## Note

KU IT recommends saving the exported `.pfx` or `.p12` file to OneDrive for security.
