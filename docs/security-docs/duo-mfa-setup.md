# Duo Multi-Factor Authentication - Setup and Device Management

## Summary

Instructions for setting up and managing Duo Multi-Factor Authentication (MFA) to secure KU accounts.

## Setup Steps

1. Log in to Single Sign-On and follow the Duo prompt.
2. Select device type (mobile phone is recommended).
3. Install the **Duo Mobile** app from the App Store or Play Store.
4. Scan the provided QR code to activate the account.

## Management

Users can add additional devices (tablets, landlines, or U2F keys) by selecting "Other Options" and "Manage devices" during login.
