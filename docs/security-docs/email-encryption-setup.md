# Email Encryption - Information and Setup

## Summary

General information on requesting and configuring email encryption at KU.

## Requesting

1. Visit [myidentity.ku.edu](https://myidentity.ku.edu).
2. Select **Email Services**.
3. Click **Request a Personal Certificate**.

## Installation

After downloading, open the certificate file and follow the OS-specific prompts to install it into your system's certificate store.

## Usage

In Outlook, use the **Options** tab and select **Encrypt** (S/MIME) to send secure messages.
