# KUMC Account - Requesting Access to KU-Lawrence Resources

## Summary

Guidance for KU Medical Center (KUMC) students and staff who need access to Lawrence campus resources.

## Setup

KUMC users must still go through the KU Online ID setup at [myidentity.ku.edu](https://myidentity.ku.edu) using their KUMC credentials.

## Identification

Employees must use their KUMC Employee ID (which starts with an "M").
