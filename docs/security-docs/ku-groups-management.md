# KU Groups - Information and Management

## Summary

Using the KU Groups tool to manage access and distribution lists.

## Access

Visit [groups.ku.edu](https://groups.ku.edu).

## Roles

Defines roles such as:

- **Maintainer:** Controls membership
- **Authorized Sender:** Can send emails to the group
- **Moderator:** Approves messages before distribution

## Visibility

Groups can be "Listed" (visible in the Global Address List) or "Unlisted."
