# ArcGIS - Access and Installation

## Summary

ArcGIS Desktop has been retired in favor of ArcGIS Online and ArcGIS Pro. This guide provides instructions on how to access ArcGIS in the cloud, install ArcGIS Pro on Windows devices, and log in to activate your installation.

ArcGIS software installations can be found at the [KU Software Web Store](https://kusoftware.ku.edu).

## Accessing ArcGIS Online Account

1. Navigate to the [KU ArcGIS Online portal](https://arcgis.com).
2. Click **Sign In**.
3. Select **The University of Kansas** (Note: Loading may take 1-2 minutes).
4. If prompted, log in with your **KU Online ID** and **password** through the Single Sign-On (SSO) page.

## ArcGIS Pro - Installation

1. Once the software is purchased, download all files associated with ArcGIS Pro from the KU Software Web Store.
2. Locate the files in your **Downloads** folder.
3. Proceed to the required setup steps below.

### Step 1: Windows Runtime Installer (Required)

1. In your downloads folder, open the file: **windowsdesktop-runtime-8.0.10-win-x64**.
2. Click **Install**.
3. Select **Yes** when prompted.
4. Once complete, click **Close**.

### Step 2: ArcGIS Installation

Follow the prompts within the primary ArcGIS Pro installer downloaded from the Software Catalog.

### Step 3: ArcGIS Patch Installation

After the main installation, run any included patch files to ensure the software is up to date.

## ArcGIS Pro - Logging In

1. Launch the **ArcGIS Pro** application.
2. Use the **Sign In** option and select the enterprise login for **The University of Kansas**.
3. Authenticate using your KU Online ID to activate the license.
