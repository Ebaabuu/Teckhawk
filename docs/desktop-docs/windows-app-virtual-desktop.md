# Windows App (Microsoft Virtual Desktop) - Installation and Use

⚠️ **Important Notice:** Do not save files directly to the virtual machine. Always save to OneDrive or personal storage to prevent data loss.

## 1. Install Windows App

### Windows

1. Go to the Microsoft Store to download the client compatible with your Windows device.
2. Open the downloaded Windows App Installer.
3. The software will automatically download and install.

### Mac

Download the Windows App (formerly Microsoft Remote Desktop) from the Apple App Store.

### iOS & Android

- **iOS:** Install "Remote Desktop Mobile" (now Windows App Mobile) from the iOS App Store.
- **Android:** Install the "Windows App" from the Google Play Store.

## 2. Configure Windows App

### Windows Steps

1. Open the Windows App and click **Sign in**.
2. Enter your KU primary email address (e.g., j123h456@ku.edu) and click Next.
3. Log in via the KU Single Sign-On page with your Online ID and password.
4. Once logged in, click **Connect** next to the "Desktop" icon.
5. Enter your KU password again and select OK to log in.

### Mac Steps

1. Open the app and click **Sign In**.
2. Enter your KU email address and log in via the KU SSO page.
3. Your available Workspaces (e.g., KU Virtual Lab) will appear.
4. Click on **Desktop** to initiate the remote connection.
5. Enter your KU password in the format j123h456@home.ku.edu when prompted.

### iOS Steps

1. Open the app and tap the plus (+) icon at the top.
2. Select **Work or School Account**.
3. Sign in with your KU email and credentials via Single Sign-On.
4. Select Desktop from the Workspaces menu and enter your KU password to connect.

## 3. Saving Files in Windows App

The Windows App environment does not support saving files directly to the host machine's drive.

- **OneDrive Integration:** Your KU OneDrive account is automatically configured and visible in the Windows App File Explorer.
- **How to Save:** Use Save As or Ctrl + S within your software and navigate to the OneDrive folder.
- **Access Anywhere:** Files saved here can be accessed from any web browser at onedrive.ku.edu.

## 4. Software Available Through Windows App

Students and employees can access specialized software through the virtual desktop, including:

- **Microsoft Project:** (Required for students needing Project access)
- **Microsoft Visio:** (Required for employees needing Visio access)
- **Minitab:** (Available for students and staff on virtual desktops)
- **School of Business Desktop:** Provides software specifically for School of Business students

## 5. Windows App - Browser

If you cannot install the dedicated app, you can access the virtual desktop through a web browser:

1. Visit virtualdesktop.ku.edu.
2. Sign in with your KU email and Online ID/Password.
3. Under the "KU Virtual Lab" heading, click on Desktop.
4. Accept the default resource selections and click Allow.
