# SAS - Installation and Renewal

## Licensing Alert

SAS software at KU is licensed from November 1st to October 31st of the following year.

- **Annual Requirement:** You must purchase a new copy of SAS yearly to obtain the newest license.
- **Grace Period:** There is a 60-day grace window where an older license will continue to work, provided the software is already installed and activated.
- **New Installations:** New installations always require the purchase of a current copy of SAS.
- **Purchase Link:** KU Software Store

## SAS 9.4 Installation

These instructions assume you have not previously installed SAS on your system.

1. **Download:** Log in to the KU Software Store and download the installer from your My Software page.
2. **Extract:** Locate the .zip file in your Downloads folder (formatted as OnlineID - SAS 9.x...). Right-click and select Extract All.
3. **Run Setup:** Open the extracted folder and right-click setup.exe, then select **Run as Administrator**.
4. **Deployment Wizard:**
   - Select your language and click OK.
   - Select **Install SAS software** and click Next.
   - Choose **Install SAS Foundation and Related Software** and click Next.
5. **Configuration:**
   - Select **64-bit Native Mode** (or 32-bit Compatibility Mode if specifically required) and click Next.
   - On the product list, click **Clear All**. English will remain selected by default; click Next.
   - Select **SAS Foundation (64-bit)** for the default file types and click Next.
6. **Finalize:** Review the deployment summary and click Start. Once completed, click Next and then Finish.

## Renew SAS License

Follow these steps to apply a new license file to an existing SAS installation.

1. **Obtain License:** Download your Renewal License from the KU Software Store.
2. **Extract:** Right-click the renewal .zip file and select Extract All.
3. **Launch Utility:**
   - Open the Start Menu, find the SAS folder, and locate **Renew SAS Software 9.x**.
   - Right-click the utility and select **Run as Administrator**.
4. **Apply File:**
   - Select your language and click Next.
   - Click Browse and navigate to the folder where you extracted your renewal license.
   - Select the appropriate .txt file (the SID file) and click Open.
5. **Complete:** Click Next, then Start to apply the new license. Click Finish once the process is complete.
