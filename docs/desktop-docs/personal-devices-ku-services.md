# Personal Devices - KU Services on Phones, Laptops, and Desktops

This guide provides specialized support instructions for students and employees using their personal equipment for KU services.

## 1. BitLocker Keys

KU Information Technology does not manage or store personal BitLocker recovery keys. However, these keys are often automatically backed up to your Microsoft account if you used a KU email address during device setup.

### How to Find Your Recovery Key

1. **Check your personal Microsoft account:** Open a web browser on a different device and visit account.microsoft.com/devices/recoverykey.
2. **Check your KU account:** If not found in a personal account, visit myaccount.microsoft.com/device-list and sign in with your KU Online ID and password.
3. **Locate Device:** Expand the specific device you are trying to unlock to expose the "BitLocker" keys.

**Important Note:** If the key is not found in either location, KU IT cannot recover it, and the operating system may need to be reinstalled, which could result in data loss.

## 2. Mac OS PRAM Reset

Resetting the Parameter RAM (PRAM) can resolve issues related to speaker volume, screen resolution, and network connectivity (such as the Jayhawk wireless network).

### Instructions

1. **Shut Down:** Completely power off the computer. Using "Restart" will not work for this process.
2. **Power On:** Press the Power button.
3. **Hold Key Combination:** Immediately press and hold **Command (⌘) + Option + P + R**.
4. **Wait:** Continue holding these four keys for about 20 seconds.
   - Note: If your Mac plays a startup sound, you can release the keys after you hear it for the second time.
5. **Finish:** Release the keys and allow macOS to finish booting. You may need to reconfigure some system preferences (like time zone or volume) afterward.
