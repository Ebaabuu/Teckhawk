# Multi Function Device (MFD) - Supply Ordering

KU utilizes an automated supply replenishment program through UDP for multifunction printers. This system uses a Data Collecting Agent to monitor consumable levels and reporting meter reads over the network.

## 1. Automated Replenishment

- **Toner and Imaging Units:** These are sent automatically when the system detects they are needed.
- **Notifications:** The designated KU contact for the device will receive an email with order details and an estimated time of arrival.
- **Identification:** Shipments from the authorized vendor will have a UDP logo on the packing slip.

## 2. Manual Ordering (Department Responsibility)

The following supplies are NOT sent automatically. Departments are responsible for ordering them as needed:

- **Staples:** Refer to your specific supply guide for the correct type.
- **Paper**
- **Waste Toner Containers**
- **Extra Supplies:** You may order extra supplies directly from UDP at any time.

Note: If you do not have a supply guide, email Teri Majors at tmajors@udpcorp.com with your equipment ID to request one.

## 3. Disposal and Recycling

To dispose of used printer toner cartridges, contact the KU Environmental Stewardship Program for recycling options.
