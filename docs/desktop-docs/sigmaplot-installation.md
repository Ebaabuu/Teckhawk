# SigmaPlot - Installation

This software requires the use of a VPN when you are not on campus for both KU-Lawrence and KUMC users. You must also have already purchased the software from the KU Software Store.

## Pre-Requisites

- **VPN Connection:** If off-campus, connect via KU Anywhere.
- **Purchase:** Ensure you have an active license from the KU Software Store.

## Installation Steps

1. **Download:** Log in to the software store, click My Software, and then click on Sigmaplot14.zip.
2. **Locate File:** Open your Downloads folder and find the downloaded zip file.
3. **Extract:** Right-click the SigmaPlot zip file and select **Extract All**.
4. **Confirm Extraction:** Ensure the box "Show extracted files when complete" is checked and click Extract.
5. **Open Folder:** Open the extracted SigmaPlot folder. Verify the type is listed as File Folder and not the compressed zip file.
6. **Run Installer:** Click on the file titled **Install_SigmaPlot14**.
   - **Warning:** DO NOT click on the SigmaplotInstaller file directly; the batch file will run it automatically.
7. **Permissions:** When Windows asks for permission to run the file, click Yes.
8. **Launch & Update:** Open SigmaPlot via the desktop icon. If prompted, click **Install Updates**.
9. **Confirm Updates:** Click Yes, then click Download.
10. **Finalize:** Click Install and then select Finish once the process is complete.
