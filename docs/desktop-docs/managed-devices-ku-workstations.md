# Managed Devices - KU Laptops and Desktop Workstations

This guide addresses common issues affecting KU-managed devices at the Lawrence and Edwards campuses.

## 1. Account Disabled & Domain Unavailable

Users who encounter messages stating "Account has been disabled" or "Domain isn't available" must contact IT for assistance.

- **Action:** Contact the KU IT Customer Service Center at 785-864-8080.
- **Alternative:** Find your specific technical support team via the TSC Directory.

## 2. Help - I Can't Do X After Changing My Password

It is common for users to lose access to services like Central File Storage (CFS) or Research File Storage (RFS), or find that their new password doesn't work locally while the old one still does.

### macOS Instructions

1. **Connect to VPN:** Use KU Anywhere to establish a connection to the KU network.
2. **Lock Screen:** Open the Apple menu and select Lock Screen (or use keyboard shortcuts).
3. **Log In:** Log back into the computer using your new password.
4. **Alternative:** Take your laptop to campus and log in while connected to the campus network.

### Windows OS Instructions

1. **Connect to VPN:** Connect via KU Anywhere.
2. **Lock Device:** Press Windows Key + L or select Lock from the account icon in the Start menu.
3. **Sign In:** Press Ctrl + Alt + Del and log in with your new password.
4. **Alternative (Laptop):** Bring the device to campus and log in while on the KU network.
5. **Alternative (Desktop):** Call 785-864-8080 to have a ticket escalated to your technical support team.

## 3. Local Logon Disabled

KU IT automatically disables any device that has not checked in with the network for 210 days.

- **Resolution:** If you encounter this, the IT help desk must open a support ticket to re-enable the device.

## 4. Software Installation

KU managed devices typically require administrator access. Use the platform-specific tools below to install approved software.

### macOS

1. Open the **Self Service** application (look for the Jayhawk icon in Launchpad).
2. Search for the desired software or browse categories in the left sidebar.
3. Click **Install**. Once finished, the button will change to "Reinstall".

### Windows OS

1. Click the **Start** button and search for **Software Center**.
2. Search for the application you need and click its icon.
3. Click **Install**.

## 5. Unblock Apps

If a specific application is being blocked by Windows security settings, follow these steps:

1. Right-click the application file and select **Properties**.
2. Navigate to the **General** tab.
3. Under the Security section at the bottom, check the box for **Unblock**.
4. Click **Apply**.
