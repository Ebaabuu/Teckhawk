# EndNote - Installation

## Overview

EndNote is a reference management tool that helps users save time, stay organized, and collaborate with colleagues during the publishing process. Active University of Kansas faculty, staff, and students can access **EndNote 2025** for free.

## Prerequisites

- **License:** You must first "purchase" the free license from the KU Software Store.
- **System:** Available for both Mac and Windows platforms.

## Installation Steps: Mac

1. **Download:** Obtain the installer from the KU Software Store after checkout.
2. **Open Installer:** Locate the downloaded `.dmg` file and double-click to open it.
3. **Run Setup:** Double-click the **Install EndNote 2025** icon.
4. **Authentication:** If prompted, enter your Mac's administrator password.
5. **Complete:** Follow the on-screen prompts to finish the installation and launch the application.

## Installation Steps: Windows

1. **Download:** Download the installation package from your Software Store account.
2. **Extract Files:** The download is typically a compressed `.zip` folder. You **must** extract all files before running the installer to ensure the license key is applied correctly.
3. **Run Installer:** Double-click the `.msi` or `.exe` file within the extracted folder.
4. **Follow Prompts:** Click through the installation wizard. You should not be prompted for a product key if the files were correctly extracted.
5. **Finish:** Once the process is complete, click **Finish** and open EndNote from your Start menu.
