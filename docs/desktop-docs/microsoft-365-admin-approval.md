# Microsoft 365 - Admin Approval for Additional Apps

Some external software components of Microsoft 365 are not included with your standard installed package. While some of these components can be added without issue, others require Admin Approval before they can be installed on your KU account.

## How to Request Approval

If you encounter an app that requires permission, follow these steps to submit a request:

1. **Click Request Approval:** Within the application or the Microsoft 365 interface, select the Request approval button.
2. **Wait for Processing:** Requests are typically processed within one business week.
3. **Follow Up:** If your request has not been approved after one business week, please contact the IT Customer Service Center.
