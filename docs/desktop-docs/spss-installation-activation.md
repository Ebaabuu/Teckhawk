# SPSS - Installation and Activation

**Alert:** This article is pending an update with specific installation and activation information for SPSS 31 and Amos 31. The instructions below cover versions 29 and 30.

## 1. SPSS 29 - Installation & Activation

### Windows Installation

1. **Download:** Log in to the KU Software Store, go to My Software, and download IBM SPSS Statistics 29.msi.
2. **Run Installer:** Double-click the file and click Next on the wizard.
3. **Agreement:** Select "I accept the terms in the license agreement" and click Next.
4. **Destination:** Use the default folder and click Install.
5. **Finish:** Click Finish with "Start IBM SPSS Statistics now" selected.

### Mac Installation

1. **Download:** Download SPSS 29.0.0.0_Mac.pkg from the KU Software Store.
2. **Launch:** Double-click the .pkg file and click Continue.
3. **License:** Click Agree to the software license terms and select Install.
4. **Authorization:** Enter your Mac administrator credentials and click Install Software.

### Activation (All OS)

1. Launch the SPSS License Authorization Wizard.
2. Select **Authorized user license** and click Next.
3. Enter the Authorization Code found on your My Software page in the KU Software Store.
4. Click Add, then Next, and then Finish once the "Authorization Successful" message appears.

## 2. SPSS 30 - Installation & Activation

### Installation

1. **Download:** Download the SPSS 30 installer from the KU Software Store.
2. **Extract (Windows):** Right-click the .zip file and select Extract All.
3. **Run as Admin:** Right-click the .exe file (Windows) or open the .pkg (Mac) and select Run as Administrator.
4. **Wizard:** Follow the prompts, accepting the license agreement and using the default destination folders.
5. **Complete:** Click Install, then Finish.

### Activation

1. Open SPSS 30 and click **Launch License Wizard**.
2. Choose **Authorized user license** and click Next.
3. Enter your SPSS 30 Authorization Code from the KU Software Store.
4. Click Add and then Next. Once the code is processed successfully, click Finish.

## 3. Activation By Email (Offline Method)

If your device cannot reach the IBM license server, follow these steps:

1. **Generate Code:** In the License Authorization Wizard, choose the Web-based e-mail option.
2. **Copy Details:** Copy the provided lock code and authorization code.
3. **Draft Email:**
   - **To:** spssls@us.ibm.com
   - **Subject:** "License Authorized"
   - **Body:** Paste the exact text generated by the wizard, including the authorization and lock codes.
4. **Apply Key:** IBM will return a license string. Paste this string into the "Enter Code" box in the wizard and click Add.
