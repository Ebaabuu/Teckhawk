# Read&Write - Access and Installation

Read&Write is no longer hosted on the KU Software Store. To install and activate the software, follow the steps below.

## Mac Installation and Activation

1. **Download:** Visit the Texthelp Read&Write for Mac page.
2. **Select Download:** Click on Try Read&Write and select Mac.
3. **Allow Download:** If prompted by your browser, click Allow.
4. **Open Installer:** Locate the read&write.dmg file in your Downloads folder and double-click it.
5. **Run Installer:** Follow the on-screen instructions in the installer window.
6. **Launch App:** Open Read&Write from your Applications folder or via Launchpad.
7. **Security Prompt:** Click Open if prompted by Apple Security.
8. **Grant Access:**
   - If prompted, select Security and Privacy.
   - Check the box to allow Read&Write access.
   - Click Continue and then OK.
9. **Accept Terms:** Select "I accept the User Terms" and click OK.
10. **Sign In:** Select **Sign in with Microsoft**.
11. **Credentials:** Enter your email address as onlineID@ku.edu. This will redirect you to the KU Single Sign-On (SSO) page.
12. **Finalize:** Log in with your KU Online ID and password, then select Accept.

## Windows Installation and Activation

1. **Download:** Visit the Texthelp website.
2. **Select Download:** Click Try Read&Write and select Windows (choose the appropriate region, such as US).
3. **Run Installer:** Once downloaded, click the file to begin installation.
4. **Accept and Install:** Click Accept and Install when the installer window appears. This may take several minutes.
5. **Launch App:** Once finished, open the Read&Write program using the icon on your desktop.
6. **Activate:** Click on the Read&Write logo in the tab at the top of your screen.
7. **Sign In:** In the browser window that opens, select **Sign in with Microsoft**.
8. **Credentials:** Enter your KU email address (j123h456@ku.edu) to be redirected to the KU sign-in page.
9. **Completion:** Once signed in, your copy of Read&Write will be fully activated.
