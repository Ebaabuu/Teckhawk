# QtiPlot - Access and Installation

QtiPlot is primarily used by Engineering students at KU. It is available for both Mac and Windows and can be purchased through the KU Software Webstore.

## 1. Mac Installation

1. **Download:** Click the file (either Intel or M1) specified for your computer.
   - Note: If you are unsure which processor your Mac uses, go to Apple Menu > About This Mac. The information is listed under Processor.
2. **Open Folder:** This will download a folder containing the installer.
3. **Run Installer:** Right-click the installer in the downloaded folder and select **Open**.
4. **Prompts:** Click Continue, then Continue again, followed by Agree.
5. **Finalize:** Click Install. Once completed, click Close.

## 2. Windows Installation

1. **Extract Files:** Right-click the downloaded file and select Show in Folder. Right-click the file again in your Downloads folder and select Extract All > Extract.
2. **Open Folders:** Open the newly created unzipped folder, then open the folder inside it (it will have the same name without your Online ID in front).
3. **Install Python:** Double-click the `python-3.10.7-amd64` file. This is a required dependency.
   - Select **Customize Installation**.
   - Check all required boxes as prompted and click Next.
   - Click Install, then Yes, and then Close once the setup is finished.
4. **Install QtiPlot:** Return to the Downloads folder and double-click the `090623_qtiplot-1.1.5-win64-python310` file.
5. **Finalize:** Follow the User Account Control prompts. Once finished, click Yes. The software is now ready for use.
