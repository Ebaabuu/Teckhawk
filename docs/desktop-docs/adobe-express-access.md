# Adobe Express - Access

## Summary

Adobe Express is offered independently of Adobe Creative Cloud and is free for all active University of Kansas students, faculty, and staff.

## Student License Activation

To use Adobe Express, students must first claim a license.

- **Claim Link:** Visit [license.ku.edu](https://license.ku.edu).
- **Processing Time:** It can take up to **24 hours** for the license to be granted after it is claimed.

## How to Log In

1. **Open the App:** Use the **Adobe Creative Cloud Desktop Application** or navigate directly to the [Adobe Express website](https://www.adobe.com/express/).
2. **Enter Credentials:** Input your **KU email address** and select **Continue**.
3. **Authentication:** If prompted, log in with your **KU Online ID** and **password** via the Single Sign-On (SSO) page.
4. **Completion:** You should then be successfully logged into the platform.
