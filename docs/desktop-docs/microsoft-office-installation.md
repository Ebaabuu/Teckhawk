# Microsoft Office - Installation and Activation

## Microsoft Project & Visio Access Information

Microsoft Project and Visio are not included in the standard installation.

- **Students:** Use the Windows App (formerly Microsoft Remote Desktop) or purchase a personal license directly from Microsoft.
- **Visio:** Available through the KU Virtual Desktop environment.
- **Employees:** Contact your technical support team for purchase. Annual costs apply and are charged to your department.

## Installation

### Mac

1. **Access Portal:** Navigate to office365.ku.edu and sign in with your KU Online ID and password.
2. **Download:** Click Install Office and select Office 365 apps to download the installer package (.pkg).
3. **Run Installer:** Open the downloaded file from your Downloads folder and double-click to start.
4. **Prompts:** Click Continue, Agree to the license terms, and then Install.
5. **Authorization:** Enter the local username and password for your Mac when prompted to allow the installation.
6. **Cleanup:** Once finished, click Close and select Move to Trash to remove the installer file.

### Windows

1. **Access Portal:** Go to office365.ku.edu and log in with your KU credentials.
2. **Download:** Click Install and more, then select Microsoft 365 apps to download the OfficeSetup.exe file.
3. **Run Setup:** Open the downloaded file. If a User Account Control (UAC) prompt appears, click Yes.
4. **Wait:** Allow the installation to complete. You will see a "You're all set!" message when finished.
5. **Open App:** Launch any Office application (like Word) from your Start menu.

## Activation

You can activate the suite using any installed Microsoft Office application.

1. **Sign In:** Open an app (e.g., Word) and click the Sign in button.
2. **Email:** Enter your KU email in the format j123h456@ku.edu and click Next.
3. **SSO:** Log in via the KU Single Sign-On page with your Online ID and password.
4. **Success:** The product will activate automatically once sign-in is complete.
