# IT Software and Services Catalog - Navigation

## Summary

The IT Software and Services Catalog provides information on software and services eligibility for students, staff, faculty, and guests at the University of Kansas. You can access the catalog directly at [technology.ku.edu/catalog](https://technology.ku.edu/catalog).

## Navigating the Catalog Website

1. **Access:** Visit technology.ku.edu/catalog in your web browser.
2. **Browsing:** Software and services are displayed in a scrollable list, ordered by title. Each entry includes:
   - **Platforms:** Compatible operating systems (e.g., Windows, macOS).
   - **Description:** A brief overview of the software or service's function.
   - **Categories:** Searchable groups for filtering.
3. **Searching and Filtering:** Use the search field at the top to find specific software by name or use the "Any" dropdown to filter by category. Click **Apply** to refresh the results.
4. **Detailed Information:** Click on a software **title/name** to view its full description page.

## Eligibility and Access

- **Affiliation Type:** Eligibility (Faculty, Staff, Student, etc.) is listed in the top-left of the product's description page.
- **Detailed Access Groups:** Specific restrictions for sub-groups (e.g., Graduate Students vs. Undergraduate Student Workers) can be found under the **Access** heading further down the page.
- **How to Download:**
  - **KU Software Web Store:** If available, you will be directed to the [KU Software Web Store](https://kusoftware.ku.edu/shop/) for free or paid downloads.
  - **Manufacturer Site:** If the software must be obtained directly from the vendor, a link will be provided.
  - **Purchasing:** If the software is not free from the manufacturer, consult your technical representative or refer to the Technology Purchases guide.
