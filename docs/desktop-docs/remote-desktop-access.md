# Remote Desktop - How to Access Remote Devices

Before connecting to a remote computer on campus, you must first establish a VPN connection if you are working remotely. You will also need the computer name and location; if you do not have this information, contact your departmental support staff.

## Connection Steps

1. **Connect to VPN:** Establish a connection to KU Anywhere on your computer.
2. **Open RDP:** Open the Remote Desktop Connection (RDP) program.
   - **Windows:** Search the Start Menu for "Remote" to find the application.
   - **Mac:** Download the Microsoft Remote Desktop software from the App Store or following Microsoft's instructions.
3. **Enter Computer Name:** Type the remote computer's name into the Computer text box.
4. **Enter Username:** Use the format `home\yourOnlineID` in the User Name field.
5. **Connect:** Click Connect and enter your KU password in the security pop-up when prompted.

Once connected, you will have full access to the remote computer's desktop as if you were physically sitting at the machine.
