# Maplesoft - Access and Installation

These instructions assume you have already obtained Maplesoft from the KU Software web store.

## Network Requirements

To install or use this software, you must be connected to one of the following networks:

- Jayhawk on the KU Lawrence Campus
- KUMC WiFi
- KU Anywhere VPN Service

## Installation Instructions

### Mac

1. Locate the downloaded Maple installer file (e.g., Maple2023.1MacInstaller.dmg).
2. Double-click the .dmg file to mount it, then double-click the Maple Installer icon.
3. Follow the on-screen prompts until you reach the License Type screen.
4. Select **Network License** and click Next.
5. Select **Single Server**.
6. Enter the License Server address: `academic.license.ku.edu`
7. Enter the Port Number: `27000`
8. Complete the installation and launch Maple while connected to a required KU network.

### Windows

1. Locate the downloaded Maple installer (e.g., Maple2023.1WindowsInstaller.exe).
2. Right-click the installer and select **Run as Administrator**.
3. Proceed through the setup wizard until you reach the License Type window.
4. Choose **Network License** and click Next.
5. Select **Single Server**.
6. Input the following server details:
   - License Server: `academic.license.ku.edu`
   - Port Number: `27000`
7. Finish the installation. Ensure you are connected to the KU Anywhere VPN or campus network before launching the application.
