# SPSS: AMOS - Installation and Activation

This software can be found in the KU Software Store. AMOS is only available for Windows and has the following system requirements:

- **Processor:** Intel or AMD x86 running at 1 GHz or higher
- **RAM:** 2 GB or more
- **Hard Disk:** 1 GB of available space
- **Monitor:** 1024 x 768 or higher resolution

## 1. AMOS 29 Installation

1. **Download:** Log in to the KU Software Store, select My Software, and download SPSS Amos 29 - Windows.msi.
2. **License Key:** Click the SPSS 2024 License Keys link while the file downloads and keep this page open for later.
3. **Run Installer:** Open the downloaded .msi file. Click Next to begin.
4. **Agreement:** Select "I accept the terms in the license agreement" and click Next.
5. **Defaults:** Keep the default settings and click Next.
6. **UAC Prompt:** Click Yes if Windows asks for permission to make changes.
7. **Crucial Step:** When the installation finishes, **deselect** "Start IBM SPSS Amos 29 now" and click Finish. Starting it before licensing will result in an error.

## 2. AMOS 30 Installation

1. **Download:** Log in to the KU Software Store, select My Software, and download SPSS 30 Amos.msi.
2. **License Key:** Click SPSS 2024 License Keys while the download finishes.
3. **Run Installer:** Open the .msi file and click Next.
4. **Wizard:** Accept the license terms, keep default paths, and click Install.
5. **UAC Prompt:** Select Yes if prompted.
6. **Crucial Step:** **Deselect** "Start IBM SPSS Amos 30 now" and click Finish.

## 3. Activation

### Standard Activation

1. **Open Wizard:** In the Windows Start Menu, go to All Apps > IBM SPSS Amos and click **IBM SPSS Amos License Authorization Wizard**.
2. **Permissions:** Select Yes if prompted by User Account Control.
3. **License Type:** Leave Authorized user license selected and click Next.
4. **Enter Code:** Paste the key from the License Keys page you kept open earlier into the Enter code box.
5. **Finish:** Click Add, then Next. Once it shows "Successfully processed all codes," click Next and then Finish.

### Activation By Email (Offline)

1. **Select Method:** In the License Wizard, choose the Web-based e-mail option.
2. **Draft Email:**
   - **To:** spssls@us.ibm.com
   - **Subject:** "License Authorized"
   - **Body:** Paste the exact text provided in the authorization window from step 2.
3. **Complete:** Click Next at the error screen, click Copy for your records, and apply the returned key once received.
