# Library Checkout Laptop - Temporary Admin Rights

Students can check out a laptop from either Watson or Anschutz Library. Staff should ensure that the student logs in to the checkout laptop while on campus prior to allowing them to leave.

## Library Staff Steps

1. Log in with your Online ID and Password at groups.ku.edu.
2. Select the group **Library Checkout Laptops Temp Admin Students**.
   - Note: You will need the role of "Group Maintainer" in that group to add a member.
3. Select the blue **Add Member** button.
4. Populate the text box with the individual you wish to add to the group and select the blue **Add** button. You can search using "Firstname, Lastname", "Lastname, Firstname", or "KUOnlineID".
5. Once you have added all desired members, select **Submit**.

## Student Steps

**Important:** Before you proceed, please visit groups.ku.edu and verify you are in the group called "Library Checkout Laptops Temp Admin Students".

If you are not in this group and it has been greater than 1 hour since checking out the laptop, please return to the library and ask them to add you. KU IT is not permitted to add students to this group.

### Instructions

1. Click the **Start** button.
2. Search for **Software Center** and select it.
3. Go to the **Applications** section.
4. Click on **Make Me Admin**.
   - **Troubleshooting:** If you do not see "Make Me Admin", go to Control Panel > Configuration Management > Actions > Machine Policy Retrieval & Evaluation Cycle and click Run Now. Wait 5 minutes and reopen the Software Center. You may need to restart the device.
5. Click on **Install**.
6. Click the Start button again and launch the **Make Me Admin** application.
7. Click on **Grant Me Administrator Rights**.
   - Note: Admin privileges expire after 10 minutes. Launch "Make Me Admin" again to regain rights if needed.
   - **Troubleshooting:** If the button is greyed out, ensure 30 minutes have passed since you were added to the group. If it still fails, open the Windows Command Prompt and run `gpupdate /force`, then restart the device.
8. When running an installer, enter your KU Online ID and password in the User Account Control (UAC) window.
