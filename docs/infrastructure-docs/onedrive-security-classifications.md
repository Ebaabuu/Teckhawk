# OneDrive: Security Storage Classifications

For questions regarding data security, contact KU IT Security at 785-864-9003 or itsec@ku.edu.

## Data Permitted for Storage

The following sensitive data types may be stored on OneDrive:

- Level 1 Federal/State Confidentiality Laws
- Personally Identifiable Information (PII)
- PII collected by research projects
- Non-disclosure Agreements (NDAs)
- Audit Reports
- Emails covering any of the above data types
