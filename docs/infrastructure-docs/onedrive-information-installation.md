# Microsoft OneDrive: Information and Installation (KU)

## Overview

Every KU user with access to the Microsoft Office Suite has 1 TB of online storage available.

## Access Methods

- **Web Portal:** onedrive.ku.edu
- **Virtual Desktop:** virtualdesktop.ku.edu
- **Desktop Client:** Available for macOS and Windows. Mobile apps are available for Android and iOS, though KU-specific instructions are not provided for mobile platforms.

## OneDrive Application: Install & Initial Setup

### Mac Installation

1. Launch the App Store on your Mac.
2. Search for OneDrive.
3. Click Get and then Install.
4. Enter your Apple ID and password to complete the installation.

### Windows Installation (or without Office 365)

1. Navigate to the Microsoft Download OneDrive page.
2. Click Download and save the installer to your computer (typically the Downloads folder).
3. Double-click the file to run the installer and wait for the sync client to finish.

## Authentication

Sign in using your KU Online ID and Password.

**Note:** If you receive a pop-up stating "There was an error running a script on this page," select Yes. This is expected behavior during the KU login process.
