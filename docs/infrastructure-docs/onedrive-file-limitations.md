# OneDrive: File Limitations

Below are common limitations for KU OneDrive for Business:

## Size Limits

- **File Size Limit:** 250 GB per file
- **Zip Limit:** 20 GB of files total within a zip folder

## Character Limits

- **Online Path:** 400 characters (includes file name)
- **Synced Path (PC/Mac):** 255 characters
- **SharePoint Online:** 260 characters

## Invalid Characters

Files cannot contain: `\ / : * ? " < > |`

## File Types

All file types are supported.
