# Servers - Requesting a New Server and Monitoring

## Summary

The following steps detail how to request the creation of a new server and monitoring services for either a new or existing server within the KU IT environment.

## Prerequisites

Before submitting a request, ensure the following requirements are met:

- **Funding:** A valid funding source (departmental index) must be identified for any associated costs.
- **Technical Contact:** A primary technical contact must be designated for the server.
- **Security Compliance:** The server must adhere to all KU IT Security policies.

## Reference Items

These resources provide additional context for server management at KU:

- **Server Lifecycle:** Information on procurement, maintenance, and retirement of server assets.
- **Service Level Agreements (SLAs):** Details regarding uptime expectations and support response times.

## Requesting a New Server

To initiate the creation of a new server, follow these steps:

1. **Submit Request:** Navigate to the New Server Request form in the KU Service Portal.
2. **Provide Specifications:** Include requirements such as Operating System (Windows/Linux), CPU, RAM, and storage capacity.
3. **Justification:** Briefly describe the intended use or project the server will support.
4. **Review:** A member of the Enterprise Systems team will contact you if further information or clarification is needed.

## Requesting Server Monitoring

Monitoring ensures the health and availability of your server. You can request monitoring for new or existing infrastructure:

- **Standard Monitoring:** Includes basic pings, CPU usage, memory utilization, and disk space alerts.
- **Custom Alerts:** Specific services or applications can be monitored upon request.
- **Process:** Use the Server Monitoring Request form to add or modify monitoring services.

## Support & Feedback

If there are issues implementing your request, the Enterprise Systems team will contact you with next steps. For immediate assistance, contact the IT Customer Service Center at 785-864-8080.
