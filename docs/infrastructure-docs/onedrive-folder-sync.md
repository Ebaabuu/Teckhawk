# Microsoft OneDrive - Configuring Folder Sync

OneDrive can sync any folder on your computer. Use these steps to select which folders you would like the OneDrive client to sync.

## Mac

1. **Open OneDrive Settings:** Select the OneDrive cloud icon in the top menu bar, select the cog icon to open settings, and then select Preferences.
2. **Account Tab:** Go to the Account tab.
3. **Choose Folders:** Select Choose folders.
4. **Sync Selection:** In the "Sync your OneDrive files to this Mac" dialog box, uncheck any folders you don't want to sync to your computer and select OK.

## Windows

1. **Open OneDrive Settings:** Select the OneDrive cloud icon in the taskbar notification area, select the cog icon to open settings, and then select Settings.
2. **Account Tab:** Go to the Account tab.
3. **Choose Folders:** Select Choose folders.
4. **Sync Selection:** In the "Sync your OneDrive files to this PC" dialog box, uncheck any folders you don't want to sync to your computer and select OK.
