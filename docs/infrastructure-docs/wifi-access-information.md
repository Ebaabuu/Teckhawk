# Wi-Fi Access and Information (Jayhawk, KU Guest, eduroam)

## Network Selection Guide

Use the appropriate network based on your affiliation and device at the University of Kansas Lawrence & Edwards campuses.

## Connection Instructions by Device

Detailed setup instructions are available for the following platforms. Note: For specific troubleshooting, refer to the Wi-Fi Troubleshooting guide.

### Android & Chromebook

- Connection typically requires your KU Online ID and password.
- Ensure the EAP method is set to **PEAP** and Phase 2 authentication is set to **MSCHAPV2** if prompted.

### iOS (iPhone/iPad)

1. Select the Jayhawk or eduroam network from your Wi-Fi settings.
2. Enter your KU Online ID and password.
3. Trust the certificate if prompted to finish the connection.

### Mac (macOS)

1. Click the Wi-Fi icon in the top menu bar and select the desired network.
2. Enter your KU credentials.
3. Accept the security certificate to establish the link.

### Windows

1. Open the Wi-Fi menu from the taskbar and select Jayhawk or eduroam.
2. Click Connect and enter your KU Online ID and password when prompted.

### Ubuntu Linux

Setup requires manual configuration of security settings, including WPA2-Enterprise and PEAP.
