# Success Factors (MyTalent) - Support

Troubleshooting for the MyTalent training platform.

## Quick Fixes

- Clear browser cache/cookies
- Use Chrome or Firefox
- Ensure monitor is in **Landscape** orientation

## Browser Settings

- JavaScript must be enabled
- Pop-up blockers must be disabled

## Mobile Device Support

- **Tablets:** Supported (iPad iOS 9.3+)
- **Smartphones:** Not recommended
