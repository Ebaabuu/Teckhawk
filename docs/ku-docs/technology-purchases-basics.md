# Technology Purchases - Basics and How to Buy

Policies for purchasing IT hardware and software.

## Two Categories

### 1. Workplace Computer Hardware
Includes laptops, monitors, mice, and similar items.

**Process:** Request via your TSC (Technology Support Center); no procurement form needed.

### 2. All Other IT
Includes servers, software, SaaS, professional services.

**Process:** Requires a **Technology Procurement Request Form**.

## Requirement

All IT purchases must be reviewed by KU IT and KU Procurement.
