# Enroll & Pay - Information and Resources

General support information for Enroll & Pay and Delegate access.

## Delegate Access

Students can create delegate accounts for others (parents, etc.).

**Important Notes:**
- `@Yahoo.com` addresses are unreliable
- `@ku.edu` addresses cannot be used for delegate accounts

**Delegate Setup Steps:**
1. Account creation
2. Set Security Question/PIN
3. Select Student

## Recovery

For delegate account verification/recovery assistance, call **785-864-8080**.

## Common Issues

- **Blocked pop-ups:** Ensure pop-ups are allowed for the Enroll & Pay site
- **"Bad User Map" error:** Avoid students and delegates using the same browser/device
