# Email - Setup and Access on a Mobile Device

## Gmail App (Android/iOS)

1. Open Gmail > Tap avatar > **Add another account**.
2. Select **Exchange and Office 365** (Recommended).
3. Enter `primaryEmail@ku.edu`.
4. Log in via the KU SSO window with your Online ID and password (Duo required).
5. *Manual settings if SSO fails:* Server: `outlook.office365.com`, Username: `onlineID@home.ku.edu`.

## Native iOS Mail App

1. **Settings** > **Mail** > **Accounts** > **Add Account**.
2. Select **Microsoft Exchange**.
3. Enter `primaryEmail@ku.edu` and a description.
4. Tap **Sign In** (not "Configure Manually").
5. Log in via KU SSO and Duo, then toggle the items you wish to sync (Mail, Calendar, etc.).

## Outlook App (Recommended)

1. Install Outlook from the Play Store/App Store.
2. Enter `primaryEmail@ku.edu` and tap **Add Account**.
3. Log in via the KU SSO and Duo.
