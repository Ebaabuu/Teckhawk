# Microsoft Outlook - Adding a Shared Account (Mail Merge) in Windows

## Summary

This process allows you to add a shared mailbox as a separate profile, which is required for features like **Mail Merge**.

## Instructions

1. Open **Control Panel** and navigate to **User Accounts** > **Mail**.
2. Click **Show Profiles...** and then click **Add**.
3. Name the profile (e.g., "Shared Mailbox Name") and click **OK**.
4. Enter the **email address of the shared mailbox** and click **Next**.
5. Select **Manual setup or additional server types** > **Next**.
6. Follow the Microsoft 365 login prompts using the shared address.
7. **Crucial:** In the Mail settings window, select **Prompt for a profile to be used**.
8. Click **Apply** and **OK**.
9. When you launch Outlook, you will be prompted to choose between your personal profile and the shared mailbox profile.
