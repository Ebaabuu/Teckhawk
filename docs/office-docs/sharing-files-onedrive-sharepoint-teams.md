# Sharing - Files and Folders in OneDrive, SharePoint, and Teams

## Summary

A quick guide on how sharing works across the entire Microsoft 365 suite at KU.

## Best Practices

- **OneDrive:** Best for personal work drafts or one-off sharing with a single person.
- **SharePoint:** Best for long-term storage and departmental access.
- **Teams:** Best for active, daily collaboration on files within a specific group.
- **Security Tip:** Always use the "Specific People" link setting when sharing sensitive data to prevent unauthorized internal forwarding.

## Sharing with External Users

External guests (e.g., KDOT, researchers) may have trouble accessing files inside a Teams channel.

1. In the Teams **Files** tab, click **Open in SharePoint**.
2. Click the three dots (...) and select **Manage Access**.
3. Click **Share** and enter the external email address.
4. **Recommendation:** Use the "People you choose" setting for sensitive data.

## Permission Levels

- **Can Edit:** User can download and change content.
- **Can View:** User can download and view but not change.
- **Can't Download:** User can view the file only in the web browser (no local saving or editing).

## Linking to OneDrive

To make SharePoint files appear in your local File Explorer (Windows) or Finder (Mac):

1. Navigate to the SharePoint library.
2. Click **Add shortcut to OneDrive**.
3. The folder will now sync to your local computer under your KU OneDrive directory.
