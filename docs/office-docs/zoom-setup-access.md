# Zoom - Setup and Access

## Account Provisioning

- All KU faculty, staff, and students are eligible for a Zoom Pro/Edu license.
- Sign in at **kansas.zoom.us** using your KU Online ID and password to activate your account.

## Joining Meetings

- **From Desktop App:** Select **Sign In with SSO**. When prompted for the domain, enter `kansas`.
- **From Canvas:** If your instructor has integrated Zoom, look for the "Zoom" link in the left-hand course navigation.

## Security Best Practices

- Use a **Waiting Room** to control who enters.
- Set a **Passcode** for all meetings.
- **Mute Participants Upon Entry** to prevent background noise disruptions.
