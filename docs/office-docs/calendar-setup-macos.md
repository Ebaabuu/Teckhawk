# Calendar - Setting Up KU Account on macOS

## Prerequisites

- An active KU email account
- A Mac running a supported version of macOS

## Instructions

1. Open **System Settings** (or System Preferences on older macOS versions).
2. Select **Internet Accounts**.
3. Choose **Add Account** and select **Microsoft Exchange**.
4. Enter your full KU email address (e.g., `jayhawk@ku.edu`) and a description (e.g., "KU Office 365").
5. Click **Sign In** when prompted to sign in to your Exchange account using Microsoft.
6. Enter your KU password and complete the **Duo Multi-Factor Authentication** prompt.
7. Select the apps you want to sync (ensure **Calendars** is checked) and click **Done**.
