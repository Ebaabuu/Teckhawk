# Zoom - Calendar Configuration

## Connecting Zoom to Outlook (Desktop)

1. Open **Outlook** and create a **New Event/Meeting**.
2. Look for the **Zoom** section and click **Settings**.
3. Click **SSO** in the bottom right.
4. Enter `kansas` in the box (for `kansas.zoom.us`) and sign in with your KU Online ID and Duo.
5. Once authenticated, Zoom is linked to your KU Microsoft account.

## Defaulting to Zoom for All Meetings

- **Windows:** File > Options > Calendar > **Meeting Providers**. Check the box for Zoom.
- **Mac:** Outlook > Settings > Calendar > **Configure** (next to "Add online meetings to all events"). Select Zoom and Save.
