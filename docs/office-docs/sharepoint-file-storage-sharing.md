# Microsoft SharePoint - File Storage, Sharing, and Linking

## Summary

SharePoint (or "myCommunity") is the recommended platform for departmental file storage and collaboration.

## Sharing Files

1. Navigate to your SharePoint site.
2. Select a file/folder and click **Share**.
3. **Important:** The default is "People in University of Kansas." For higher security, select **Specific people** and enter their emails.
4. Choose between **Can edit** or **Can view**.

## Linking to OneDrive

To see SharePoint files in your local File Explorer, click the **Sync** button at the top of the SharePoint library.
