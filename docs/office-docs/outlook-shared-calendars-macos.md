# Microsoft Outlook - Unable to Add Shared Calendars on macOS

## Problem

The "New Outlook" mode on Mac sometimes prevents adding shared calendars via the "Open Shared Calendar" menu.

## Solutions

- **Option 1 (Legacy Mode):** Click **Outlook** in the menu bar and uncheck **New Outlook**. Revert to the legacy interface to add the calendar, then switch back.
- **Option 2 (Accept via Email):** Have the owner re-share the calendar. Open the invitation email in Outlook and click **Accept**. This should force the calendar into your Shared Calendars list.
- **Option 3 (Mobile App):** Add the shared calendar using the **Outlook Mobile app** on your phone. It will automatically sync to your Mac client.
