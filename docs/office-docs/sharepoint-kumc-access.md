# Microsoft SharePoint - KUMC Student Access

## Crucial Note

KUMC (Medical Center) and KU-Lawrence use separate SharePoint environments.

- **KUMC Site:** `kumc.sharepoint.com`
- **KU-Lawrence Site:** `kansas.sharepoint.com`

KUMC students attempting to access Lawrence resources must often set up a **KU Online ID** separately if they have not done so already. Login credentials for one do not automatically grant access to the other.
