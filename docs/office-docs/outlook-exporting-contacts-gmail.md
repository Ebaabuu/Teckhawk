# Microsoft Outlook Web App - Exporting Contacts to a Gmail Account

## Summary

This is for personal contacts only. Copying the Global Address List (GAL) is a violation of KU policy.

## Instructions

1. Log in to [outlook.office.com](https://outlook.office.com).
2. Click the **People** icon (bottom left).
3. Select **Your Contacts** and click **Manage** > **Export contacts**.
4. Choose "All contacts" and download the `.csv` file.
5. Go to [contacts.google.com](https://contacts.google.com).
6. Click **Import** > **Select file** and upload the `.csv`.
