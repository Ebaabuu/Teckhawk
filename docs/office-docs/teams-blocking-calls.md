# Microsoft Teams - Blocking Calls

## Summary

Prevent specific numbers or people from calling you through the Teams interface.

## Instructions

1. In Teams, click the **three dots (...)** next to your profile picture and select **Settings**.
2. Go to the **Privacy** tab.
3. Click **Edit blocked contacts**.
4. Type the name or number you wish to block and select it.
5. You can also block "No caller ID" calls by toggling the **Block calls with no caller ID** switch.
