# Microsoft Outlook - Recovering Deleted Items and Emails

## Summary

Items deleted from the "Deleted Items" folder can still be recovered for up to 14 days.

## Instructions

1. Open Outlook and select the **Deleted Items** folder.
2. At the top of the message list (or under the **Home** tab), click **Recover Deleted Items From Server**.
3. A window will appear listing items that were recently emptied from your trash.
4. Select the items you wish to restore.
5. Ensure **Restore Selected Items** is selected and click **OK**.
6. The items will return to your **Deleted Items** folder.
