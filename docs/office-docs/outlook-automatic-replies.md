# Microsoft Outlook - Setting Automatic Replies (Out-Of-Office)

**Note:** Outlook only sends **one** automatic reply per sender. You must toggle the rule off and on again to "reset" it for a new trip or event.

## Outlook Desktop (Windows)

1. Click **File** > **Automatic Replies (Out of Office)**.
2. Select **Send automatic replies**.
3. Set a date/time range if desired.
4. Compose your message in the **Inside My Organization** tab (for KU users) and the **Outside My Organization** tab (for everyone else). Click **OK**.

## Outlook Web App (OWA)

1. Settings (gear icon) > **Mail** > **Automatic Replies**.
2. Toggle on **Automatic replies on**.
3. Enter your message(s) and click **Save**.
