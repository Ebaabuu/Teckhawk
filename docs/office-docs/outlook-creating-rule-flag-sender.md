# Microsoft Outlook - Creating a Rule to Flag a Specific Sender

## Summary

This allows you to automatically apply a follow-up flag to emails from a specific person so they stand out in your inbox.

## Instructions (Outlook Desktop)

1. Right-click an existing email from the sender you want to flag.
2. Select **Rules** > **Create Rule...**
3. In the "Create Rule" window, check the box: **From [Sender Name]**.
4. Click the **Advanced Options...** button.
5. In the Rules Wizard, click **Next** (since the sender is already selected).
6. In Step 1 (Select actions), check the box: **flag message for follow up at this time**.
7. In Step 2 (Edit the rule description), click the underlined phrase **"follow up at this time"**.
8. Choose your flag type (e.g., "Today") and click **OK**.
9. Click **Finish**, then check **Run this rule now on messages already in the current folder** and click **OK**.
