# Microsoft Teams - Adding Non-KU External Members to Shared Channels

## Summary

Shared channels allow you to collaborate with external partners (e.g., consultants or researchers) without them needing a KU account.

## Requirements

- The external user must have a **Microsoft 365 account** at their own organization.
- You must be the **Owner** of the Team.

## Process

1. Open the Channel menu (...) and select **Manage channel**.
2. Go to the **Members** tab and select **Add members**.
3. Type the full email address of the external guest.
4. Note: External access is restricted by default for certain high-security departments. If the email is rejected, contact **itcsc@ku.edu**.
