# Departmental Shared Mailbox - Sending Email

## From Outlook for Windows

1. Open a **New Email**.
2. If the "From" field is not visible, go to the **Options** tab and select **From**.
3. Click the **From** button and select **Other Email Address**.
4. Type the address of the shared mailbox or select it from the Global Address List.
5. Compose your message and click **Send**.

## From Outlook Web App (OWA)

1. Click **New Mail**.
2. Click the **More options (...)** menu and select **Show From**.
3. Right-click your name in the From field and select **Remove**.
4. Type the shared mailbox address and send as usual.
