# Simplelist (Formerly Mailman) - Management, Setup, and Subscriptions

## Summary

Simplelist is a mailing list service used for informal groups, clubs, or organizations that include external (non-KU) members. It replaces the legacy "Mailman" system.

## Key Differences

| Feature | KU Group List | Simplelist |
| :--- | :--- | :--- |
| **Membership** | KU email addresses only | KU and non-KU addresses |
| **Management** | Managed via Outlook/AD | Web-based self-service interface |
| **Ideal For** | Official departments/classes | Informal clubs and interest groups |

## Management & Setup

- **Requesting a List:** Complete the Group List request form in the IT Service Portal. The system will determine if you need a Group List or a Simplelist based on your requirements.
- **Subscriber Management:** Owners can add or remove subscribers, approve moderated messages, and manage bounce scores through the Simplelist web portal.
