# Microsoft Outlook - Removing and Re-adding Outlook Profile in Windows

## Summary

This process resets your Outlook configuration without deleting your actual email data on the server. Create the new profile *before* deleting the old one to avoid data loss.

## Instructions

1. Press the **Windows Key** and type **Control Panel**.
2. Search for **Mail** (or check under **User Accounts**).
3. Click **Show Profiles...**.
4. Click **Add**, name the new profile (e.g., "Outlook New"), and click **OK**.
5. Enter your KU email details and click **Next**. Sign in via KU SSO if prompted.
6. Once finished, ensure **"Prompt for a profile to be used"** is selected.
7. Close and reopen Outlook, then select the new profile.
8. Once verified, you can return to the Mail settings and **Remove** the old profile.
