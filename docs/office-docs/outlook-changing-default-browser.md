# Microsoft Outlook - Changing Default Browser Links in Emails

## Summary

Newer versions of Outlook default to opening links in **Microsoft Edge** regardless of your system's default browser.

## How to Change

1. In the Outlook Desktop app, go to **File** > **Options**.
2. Select the **Advanced** tab on the left.
3. Look for the section: **File and browser preferences**.
4. In the dropdown for **Open hyperlinks from Outlook in:**, select **Default Browser** (or your preferred browser).
5. Click **OK**.
