# Microsoft Outlook - Exporting KU Email to a .pst Data File

## Summary

Exporting to a `.pst` file creates a local backup of your emails, calendar, and contacts.

## Instructions (Windows)

1. Open Outlook and go to **File** > **Open & Export** > **Import/Export**.
2. Select **Export to a file** and click **Next**.
3. Select **Outlook Data File (.pst)** and click **Next**.
4. Select the account or specific folder you want to export. To export everything, select your email address at the top and ensure **Include subfolders** is checked. Click **Next**.
5. Click **Browse** to choose where to save the file and name it.
6. Click **Finish**. (You may be prompted to add an optional password to the file; if you don't want one, just click **OK**).
