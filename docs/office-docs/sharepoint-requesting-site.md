# Microsoft SharePoint - Requesting a Site

## Summary

KU faculty and staff can request new SharePoint sites for departmental use or project collaboration.

## How to Request

1. Visit the KU IT SharePoint Request Page at services.ku.edu.
2. Complete the **Site Request Form**.
3. You will need to provide:
   - **Site Name:** A clear, descriptive title.
   - **Site Owners:** At least two owners are required for departmental sites.
   - **Site Type:** Communication site (for broad info sharing) or Team site (for collaboration).
4. Requests are typically processed within 2–3 business days.
