# KU Website - Submitting a Change Request

## Summary

For websites managed by KU IT, all change requests must be submitted via email to the **KU IT Web Services** team.

## Submission Details

- **Email to:** `webservices@ku.edu`
- **Required Information:**
  - Organization Name
  - Specific URL(s) of the page(s) requiring changes
  - A clear description of the requested changes for each page
  - Attachments for any new images (sized appropriately) or documents
