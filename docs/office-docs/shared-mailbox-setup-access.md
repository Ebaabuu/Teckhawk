# Departmental Shared Mailbox - Setup and Access

## Accessing via Outlook (Windows/macOS)

In most cases, if you have been granted "Full Access" permissions, the shared mailbox will automatically appear in your folder list in Outlook after restarting the application.

## Manual Addition (if it doesn't appear)

1. Go to **File** > **Account Settings** > **Account Settings**.
2. Select your account and click **Change**.
3. Select **More Settings** > **Advanced**.
4. Click **Add** and type the name of the shared mailbox.
5. Click **OK** and restart Outlook.
