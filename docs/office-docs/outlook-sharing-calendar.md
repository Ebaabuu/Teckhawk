# Microsoft Outlook - Sharing Your Calendar

## Internal Sharing (KU/KUMC Users)

1. Open your Calendar in Outlook.
2. Click **Share Calendar** (or **Permissions**).
3. Click **Add** and search for the user in the address book.
4. Set permission levels:
   - **Can view when I'm busy:** Shows time blocks only.
   - **Can view all details:** Shows titles and descriptions.
   - **Can edit:** Allows the user to create/change events.
5. Click **Apply**.

## External Sharing

KU only permits sharing "Free/Busy" info with external services. You can generate an ICS link via OWA Settings > **Calendar** > **Shared calendars** > **Publish a calendar**.
