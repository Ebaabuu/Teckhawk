# Microsoft Outlook - Identifying Your App's Version

## Instructions (Windows)

1. Open Outlook.
2. Click **File** in the top-left corner.
3. Select **Office Account** (or **Account**).
4. Look under **Product Information**. You will see the version name (e.g., "Microsoft 365 Apps for enterprise").
5. For specific build numbers, click the **About Outlook** button.

## Instructions (macOS)

1. Open Outlook.
2. Click **Outlook** in the top menu bar (next to the Apple icon).
3. Select **About Outlook**. A window will appear showing the version and build number.
